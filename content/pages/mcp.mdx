<H2>Usage Examples</H2>

<H3>Get All Posts</H3>

<p>
  Try it with Postman (MCP beta) or Claude Desktop&apos;s MCP console to fetch the
  full list of posts.
</p>

```bash
POST /api/mcp
{
  "method": "tools/call",
  "params": {
    "name": "get_all_posts",
    "arguments": {}
  }
}
```

<H3>Search Content</H3>

<p>
  Use the same tooling setup to issue a search request and filter by content
  type.
</p>

```bash
POST /api/mcp
{
  "method": "tools/call",
  "params": {
    "name": "search_content",
    "arguments": {
      "query": "React",
      "contentType": "posts"
    }
  }
}
```

<H3>Get Analytics</H3>

<p>
  Send this request from Postman or Claude Desktop to pull analytics details
  for a specific slug.
</p>

```bash
POST /api/mcp
{
  "method": "tools/call",
  "params": {
    "name": "get_content_analytics",
    "arguments": {
      "contentType": "posts",
      "slug": "specific-post-slug"
    }
  }
}
```
