title: Secrets of building robust UI components
---

React UI components can be written in many different ways and patterns. As your app grows, it’s common to increasingly introduce complexity and add up logic to UI components. In the sea of learning materials online, it’s a challenge to decide on a solution that lasts for your project.

Let’s briefly review typical steps and case studies when building UI components and unveil the secrets of writing the most resilient TypeScript and component patterns for React to build UI components at scale.
